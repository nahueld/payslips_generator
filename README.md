# myob_payslips_generator
Generates payslips based on employee details

#Dependencies
- node (v6.9.5)
- typescript (``npm install -g typescript``)

#Building
- Run tests: ``npm run test``
- Build: ``npm run build``
- Start local server: ``npm  run start``

# Bugs
